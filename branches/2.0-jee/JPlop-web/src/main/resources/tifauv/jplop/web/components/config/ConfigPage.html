Configuration des coincoins

Coincoins compatibles BoardConfigSpec

L'URL d'auto-configuration est http://localhost:8080/jplop/discover.

Si votre coincoin n'a pas de système d'auto-configuration, débrouillez-vous pour lui faire manger ça :

<site name="JPlop" title="JPlop" baseurl="http://localhost:8080/jplop" version="1.0">
	<account>
		<login method="post" path="/logon">
			<field name="login">$l</field>
			<field name="password">$p</field>
		</login>
		<logout method="get" path="/logout"/>
	</account>
	<board name="board" title="Tribune">
		<backend path="/backend" public="true" tags_encoded="false" refresh="60"/>
		<post method="post" path="/post" anonymous="true" max_length="TODO">
			<field name="message">$m</field>
		</post>
	</board>
</site>

wmcc

Ajoutez les lignes suivantes dans le fichier .wmcoincoin/options de votre dossier personnel :

board_site:                "JPlop"
.backend_flavour:          2
			
.palmipede.userlogin:      login
			
.backend.url:              http://localhost:8080/jplop/backend
.post.url:                 http://localhost:8080/jplop/post
.tribune.delay:            60
.palmipede.msg_max_length: 512

Ajoutez également la ligne suivante dans le fichier .wmcoincoin/options.auth de votre dossier personnel :

"JPlop" cookie: "JSESSIONID=DEADBEEF"
